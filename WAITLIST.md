# Early-adopter waitlist

If you'd like to be among the first companies to deploy LIQAA Meet at scale, drop us a line.

## What you get

- **Direct line to the maintainers** for setup help and bug-fix prioritisation
- **Early access to v0.4** features (Whisper WASM captions, recording, E2E encryption)
- **Custom feature consideration** if your use case is interesting and shareable
- **Logo on the README's "Trusted by" list** when you're public-ready

## What we ask

- That you're genuinely planning to deploy this — not a hobby project hoarding logos.
- That you can share the use case (so we can build it into examples and case studies).
- Honest feedback. If something is broken, tell us. If something is wonderful, tell us so we can ship more of it.

## How to sign up

Email **[partners@tkawen.com](mailto:partners@tkawen.com)** with:

- Your company / organisation name
- The use case (support, telehealth, classroom, sales, internal, …)
- Approximate scale (concurrent participants)
- Whether you'll self-host or use the hosted instance
- Whether you're OK with us publishing your name as an early adopter

We respond within 48 hours. No pre-paid contracts, no NDAs to start.
